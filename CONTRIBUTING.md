# Como Contribuir

Obrigado pelo interesse em melhorar este workshop! Contribuições são bem-vindas.

## O que podes contribuir

- **Correções** — erros de conteúdo, factos desatualizados, ligações quebradas
- **Melhorias** — clareza do guia, novos exemplos, melhores analogias
- **Traduções** — versões do workshop noutros idiomas
- **Novos desafios** — ficheiros alternativos com hashes para o mini-desafio final
- **Ferramentas** — melhorias nos scripts de configuração da VM

## Como contribuir

1. Faz **fork** do repositório
2. Cria um branch descritivo: `git checkout -b melhoria/guia-alunos`
3. Faz as alterações e confirma com uma mensagem clara
4. Abre um **Pull Request** com uma descrição do que mudou e porquê

## Boas práticas

- Todo o conteúdo deve estar em português europeu (pt-PT)
- Mantém o tom acessível — o público são jovens do ensino secundário
- Não incluas dados reais de pessoas (e-mails, palavras-passe, etc.)
- Testa os scripts antes de submeter

## Dúvidas

Abre uma [issue](../../issues) para discutir ideias antes de começares a trabalhar numa alteração maior.
